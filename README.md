## Tech Stack

- React
- TypeScript
- TailwindCSS
- Framer-Motion
- Chart.js

## Run Locally

Clone the project

```bash
  git clone https://github.com/gysenlionel/pokedex
```

Go to the project directory

```bash
  cd pokedex
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
